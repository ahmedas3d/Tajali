## Summary

Deliver the complete visual theme system for تَجَلِّي: fix WCAG AA contrast on gold text tokens, implement a user-togglable light/dark mode with `shared_preferences` persistence via Riverpod, and wire the theme toggle into the HomeScreen AppBar. The four theme files are already scaffolded from Phase 1; this phase corrects the accessibility gap, adds the `goldText` token, implements `StorageService`, and adds the `ThemeNotifier` provider + toggle UI.

## Implementation Steps

### Step 1 — Add `goldText` and shadow tokens to `AppColors`

Add `goldText` (dark gold ~`#7B5F00`, verified ≥4.5:1 on `backgroundParchment`) alongside the two card shadow constants that are already in `app_theme.dart` as anonymous inline values — move them to named tokens for discoverability.

Files: `lib/app/theme/app_colors.dart`

### Step 2 — Update text styles to use `goldText`

`heading1` and `goldLabel` currently reference `AppColors.gold` which fails WCAG AA on light backgrounds. Change both to `AppColors.goldText`.

Files: `lib/app/theme/app_text_styles.dart`

### Step 3 — Implement `StorageService`

Replace the no-op stub with a real `SharedPreferences`-backed implementation. `initialize()` calls `SharedPreferences.getInstance()`. `write()` calls `prefs.setString()`. `read()` calls `prefs.getString()`.

Files: `lib/shared/local_storage/storage_service.dart`

### Step 4 — Create `ThemeNotifier` provider

`AsyncNotifier<ThemeMode>` that reads the stored theme on `build()` via `StorageService`, defaults to `ThemeMode.light`, and exposes `toggle()` / `setMode()`. Persist on every state change.

Files: `lib/core/providers/theme_provider.dart` (new)

### Step 5 — Wire theme provider into `TajaliApp`

Replace the hardcoded `themeMode: ThemeMode.light` with `ref.watch(themeProvider).valueOrNull ?? ThemeMode.light`.

Files: `lib/app/app.dart`

### Step 6 — Add toggle icon to `HomeScreen` AppBar

Add an `IconButton` (moon when light, sun when dark) to `HomeScreen`'s AppBar `actions`. Tap calls `ref.read(themeProvider.notifier).toggle()`. Convert `HomeScreen` to `ConsumerStatelessWidget`.

Files: `lib/features/home/presentation/home_screen.dart`

### Step 7 — Write widget tests

Two test files:
1. `theme_toggle_test.dart` — pumps app, taps toggle, verifies `ThemeMode` changes; relaunches, verifies persistence
2. `wcag_contrast_test.dart` — asserts `goldText` contrast ratio ≥ 4.5:1 against `backgroundParchment` using the luminance formula

Files: `test/widget/`
