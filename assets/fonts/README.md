# Font Files Required

Place the following font files in this directory before building:

| File | Family | Weight | Purpose |
|------|--------|--------|---------|
| `Amiri-Regular.ttf` | Amiri | 400 | All UI text (headings, body, labels) |
| `Amiri-Bold.ttf` | Amiri | 700 | Bold UI text |
| `AmiriQuran.ttf` | AmiriQuran | 400 | Quranic Arabic text only |

Download from the [Amiri font project](https://github.com/alif-type/amiri/releases).

Font files are not committed to the repository. Each developer must obtain and place them here manually.
